# 特朗普政府式主动管理：对华为等中国科技企业的管理实践总结

## 一、背景

对近四年以来（2017年~2021年），在特朗普政府对中国科技企业管理中出现的主动管理、计划外退出等案例进行整理和复盘，提炼出对华科技企业管理中的经验教训，指导我们在识别、制裁、监管、清退等各环节的改进，尤其在国家安全审查环节需要强化一些评估要素，持续优化审查环节的标准和管理过程中差异化的制裁方案，提升对华科技企业的有效管控率。

## 二、问题/事件描述

特朗普政府执政期间（2017年~2021年），出现约9例中国科技企业被制裁（含主动管理）情况，涉及通信、互联网、半导体等多个关键领域，一定程度维护了美国的技术安全，本文对案例进行整理和提炼，识别风险因素，提炼共性原因，思考管理方案。

## 三、分析与对策

- **华为技术有限公司**
  - 基本数据：1987年成立，民营企业，全球化运营，在美业务4年，世界500强企业，通信设备专业
  - 特征：自主研发了5G技术就通过技术优势进入美国市场
  - 市场准入复盘：安全审查环节第一次引起关注，但因市场需求紧迫，运营商对设备补充比较迫切，再三游说后，市场准入勉强获得
  - 运营中结果：运营中与政府交流不多，业务几年积累到一些市场份额就准备进一步扩张。2019年5月被列入实体清单主动管理

- **中兴通讯股份有限公司**
  - 基本数据：1985年成立，国有控股，全球化运营，在美业务3年，技术实力较好，专业对口，管理层多为国企背景
  - 特征：企业沟通积极，合规思路清晰
  - 市场准入复盘：安全审查意见较好，合规沟通比较积极，财务审计正常。当时美国市场竞争压力大，需要补充新的供应商。审查正常通过
  - 运营中结果：进入市场后就经常出现合规问题（向伊朗出口等），业务开展后就隔三差五违反制裁规定，整体合规率低于50%标准，无法有效维持业务。2018年4月被主动管理

- **字节跳动（TikTok）**
  - 基本数据：2012年成立，民营企业，TikTok运营2年，知名互联网企业，有丰富的算法开发经验
  - 特征：对美国市场扩张非常关注，有成功的产品运营经验，用户增长能力很好，技术实力强，发展迅速
  - 市场准入复盘：CFIUS审查正常通过
  - 运营中结果：运营中技术能力较好，一开始用户增长和市场表现都比较好。后来感知到数据安全监管比较严后开始在透明度上有所保留，对监管部门的配合也不够积极，数据本地化要求也无法完全满足。2020年8月面临主动管理

- **大疆创新科技有限公司**
  - 基本数据：2006年成立，民营企业，在美业务3年
  - 特征：个人消费级无人机技术和军用技术都比较匹配。企业有丰富的技术经验，产品技术非常匹配且有全球市场经验
  - 市场准入复盘：技术审查正常通过
  - 运营中结果：近期因军民两用技术及数据安全问题导致业务中无法获得政府信任，市场表现不足预期。2020年12月被主动管理

- **腾讯控股有限公司（微信支付）**
  - 基本数据：1998年成立，在美业务2年，知名互联网企业，有丰富的移动支付经验
  - 特征：技术和市场拓展意愿都比较好，产品技术匹配，用户沟通积极
  - 市场准入复盘：金融监管审查正常通过
  - 运营中结果：业务中成果和用户增长都良好。因近期与美国本土支付企业竞争加剧，且数据处理涉及跨境传输，因金融数据安全整体考量，导致监管成本超过2倍预期，无法接受而计划退出

- **阿里巴巴集团（支付宝）**
  - 基本数据：在美业务2年
  - 特征：技术和业务拓展意愿都比较好，金融科技匹配，监管沟通积极
  - 市场准入复盘：CFIUS审查通过
  - 运营中结果：因近期业务扩张，涉及敏感的金融基础设施领域，距离核心金融监管要求差距较大，合规成本过高无法接受而主动退出

- **海康威视数字技术股份有限公司**
  - 基本数据：2001年成立，国有控股，在美业务半年，知名安防企业，8年国际市场经验
  - 特征：技术满足市场要求，产品经验和技术满足要求，但在人权议题上沟通能力一般
  - 市场准入复盘：技术审查勉强通过
  - 运营中结果：业务中安排了政府采购类项目，该业务需要大量的合规审查和人权尽职调查。因其在新疆业务的透明度比较差，与国会也不怎么交流。且在国内市场，主要是接收政府被动安排的项目。主动进行人权风险管理意识低。透明度弱，无法满足合规需求

- **科大讯飞股份有限公司**
  - 基本数据：1999年成立，民营企业，技术实力比较好，在美业务半年
  - 特征：AI技术比较好，语音识别专业对口。技术实力比较好，研发期间积极参加各种AI竞赛，很享受做技术创新的过程
  - 市场准入复盘：技术专业对口，虽然没有美国本土化经验，但技术优秀，AI算法测试满分，团队沟通比较开朗乐观，技术审查正常通过
  - 运营中结果：实际业务中涉及语音数据处理，有明显的数据安全合规畏难情绪。安排的合规要求不能独立完成。2019年10月被主动管理

- **商汤科技开发有限公司**
  - 基本数据：2014年成立，之前有学术研究背景。在美业务4个月。知名AI企业
  - 特征：技术沟通积极，掌握的AI算法较多，融资情况一般，希望通过美国市场获取收入。技术实力较好，估值较好，人工智能专业对口，第一次进入消费级市场因产品不对口，后来转向企业级AI服务成功
  - 市场准入复盘：技术沟通积极流畅专业。专注人工智能开发。主要业务为AI技术服务，收入还可以，也是前沿科技类的。技术审查正常通过
  - 运营中结果：业务前几个月积极性、技术成果都不错。几个月后，因涉及敏感的人脸识别项目，监管部门需要签署数据安全不外泄的协议。企业觉得无法保证数据完全本地化，因此主动退出。企业在有一定技术优势的基础上，还是希望保持灵活的数据处理模式，因此遇到严格协议要求就选择主动退出

## 四、处理结果

综上案例数据，9家企业中有5家是主动管理，4家为计划外退出。

## 五、经验和教训

综合上述数据，对中国科技企业的管理数据进行整理，提炼出一些市场准入环节中的重点关注点，需要针对这些特征，在审查环节中加大重点考核。

### 5.1 国有企业背景情况

1. 重点关注政策承压能力。因新一代国有企业，且政府支持力度还可以的，对制裁的承压和传统民企不一样。同样的制裁措施，民企可能直接退出，国企可能就无法轻易妥协，进而导致对抗升级。在审查环节中，对于国有背景的企业，需要在国家安全审查环节加强一下压力测试。
2. 关注日常行为和市场目的。审查中要重点关注企业是否对美国市场真的感兴趣。一些国有企业进入美国也是由于政府要求，或战略布局需要而进入，日常经营重心也不会强化美国本土化运营，而是全球化布局。这种情况存在遇到一些市场中的高合规要求，就难免有抵触情绪。
3. 关注技术领域，关注其发展动力。有些企业兴趣在5G等敏感领域、或者是想体验一下美国的市场环境等，而不是真的对长期合规经营有兴趣。在其体验过程完成后，就会选择其他的市场去拓展。
4. 关注其合作伙伴，关注其是否能较好融入美国商业生态。大部分国有企业当前经营模式都比较独立，呈现出一部分企业沟通非常积极流畅，一部分就非常保守封闭。前者会掩盖合规的不足，给人开放透明的感觉，后者在经营中会和本土企业融入困难，尤其难适应美国高透明度的监管方式，导致不能满足预期。
5. 关注其政府关系和对企业的发展规划。一些政府支持力度和资源配置不错的企业，未来如何发展其实政府是有要求的，在审查中一般不表现出来。时机成熟的时候就会按政府意图行动，在审查环节中需要加强这一重要要素的考察。

### 5.2 技术转移风险

1. 安全审查加强考核。涉及技术转移的企业多多少少是技术相对敏感的，前面其他国家的监管往往以没有管辖权为理由没有深入审查，很可能是审查不够严格。存在技术或数据安全弱项，需要加强安全审查。
2. 审查增加技术评估内容。一般企业自我描述相对保守，技术能力有时候点到为止，这个在国家安全审查环节要增加一些技术专家评估。
3. 需要坚持标准。不能因为市场需求急迫或者贸易指标需要快速达成等因素就降低审查标准。

### 5.3 知名企业关注

1. 严控跨领域扩张。对于企业实力可以，但业务领域不对口的或跨度非常大的，需要强化审查，毕竟隔行如隔山，一些基本的合规概念都不清晰，会导致经营中非常困难，甚至无法适应。尤其是传统制造业，基本不可能快速转换到高科技服务业务。
2. 关注其同行是否有在美国成功运营，如果有，需要关注其同行在美国的业务性质，合规情况怎么样。大部分知名企业，如果同行在美国，尤其有同行以完全合规形式运营成功，而自己面临更严格审查，对比起来会有很大的不公平感。即使当前通过审查进入市场，也存在很大合规风险。
3. 关注其市场目的。中国企业相比美国本土企业，从监管成本和政治风险上都有一定差距。进入美国市场可能是权宜之计。要么尽快希望通过技术转移获得核心技术，要么作为临时跳板，经营不久后转向其他市场。且经常会出现技术获取不顺利，就会出现合作失望、投入积极性降低、挑选项目、甚至和监管部门有一些摩擦等，最终导致不欢而散。
4. 技术实力特别强的企业。比如技术也好，市场表现也很好，行业领先的企业，整体是比较自信的，在当前的美国市场上如果不能快速获得技术优势、或不能从本土合作伙伴身上学习到很多先进经验的情况下，也很快对当前的合作表示不满，从而重新选择市场。
5. 要关注审查时的态度，知名企业或技术领先的公司审查时往往会有一股子傲气（技术领先不愁市场的审查态度），对审查配合环节往往被动，而没有迫切想获得市场准入机会的谦逊。这种傲气在经营中会导致合规配合不主动，监管要求不遵从等管理风险。

### 5.4 国际关系考量

1. 要关注其母国情况，比如即将面临贸易摩擦升级。如果母国政策立场强硬，且与美国分歧较大，大概率会因此而被迫退出或转移业务。相比其他国家企业的经营身份，中国企业在面对政治和商业的时候，往往会以国家整体利益为重。
2. 关注股东是否有政府背景。企业股东有政府背景往往容易受到政策影响，不少企业可能是对美国整体市场环境感兴趣，先进入后进行一段探索，之后再决策，政府的政策导向往往更有决定性和影响力。
3. 管理层是否有政府工作背景。一般有高管在政府部门工作过，往往会有明确的政策导向影响（即通过政府关系能较好的获得支持以及政策倾斜）。

### 5.5 企业发展历程分析

1. 有过快速扩张经历。这类企业在相对宽松的环境下发展习惯了，很难适应严格的合规要求，如果有机会，肯定还会选择更为宽松的监管环境。
2. 有参加各种国际竞争经历。这类企业，大部分战略相对激进，对传统稳健的美国市场运营可能会不适应，或者会感觉限制其发展潜力。且在各种国际竞争中，有一定活跃的合作网络，其他市场的机会可能也很多。
3. 有接受过政府补贴经历。这类企业大部分因为政府产业政策要求，或企业受政策导向影响，选择更为依赖政府支持的发展模式。市场化经营既不是其主要模式，更不是政府规划的方向，因此即使进入美国市场，也会存在经营模式风险。
4. 之前有过国际化经历。之前的国际化经历会影响企业未来经营的方法和策略，多数情况下与美国的监管要求会有冲突，甚至觉得美国的合规要求过于严格，导致在经营中出现配合不顺畅，主动透明意识淡薄，信息披露质量差等问题，导致监管成本比较高。
5. 关注其盈利的模式。有过政府支持经验的企业往往对补贴特别依赖，对市场化盈利等长期才能实现的目标有不确定感，会对成本控制非常敏感。有时候会出现多次谈判才能达成合规协议的情况，如果在合规成本和监管要求上，无法达到其对盈利的预期，可能就会出现退出因素。

## 六、总结

中国科技企业是美国技术安全管理的重要对象，识别比监管更重要，通过持续累积经验教训、复盘提炼，指导我们在企业审查环节更加精准识别到最需要管控的对象，发挥好制裁措施的针对性和有效性，实现国家安全和经济利益的双重保障。

本文根据特朗普政府近四年的多个案例，提炼出中国科技企业被制裁（含主动管理）群体的一些特征，尤其在国有背景、知名企业、技术领先、发展历程复杂等几个维度上识别出一些共性。需要在国家安全审查环节特别关注和加强识别。
